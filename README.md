# PokeList

### 🔥 Descrição
PokeList é um site com a informação de todos os Pokémon, nele você pode ver informações como: 
- número da pokédex;
- sprites;
- evoluções;
- tipagem;
- fraquezas e resistências;
- status;
- entrada da pokedéx.
Além disso, você pode filtrar os pokémon por tipo.

### 🎯 JUSTIFICATIVA
Queremos fazer esse site para ampliar o conhecimento sobre os pokémon, tanto antigos quantos os novos para novas pessoas, como públicos adultos, adolescentes e crianças, e queremos também melhor desempenho.
